# 🐍 Activity 2 - Snake Game

Este repositorio contiene una versión modificada del juego clásico Snake, tomado del paquete `freegames` de Grant Jenks. Como parte de la actividad del curso, se realizaron modificaciones al código original para agregar dos nuevas funcionalidades, cumpliendo con el rol de Owner y Fork.

## 📁 Contenido del repositorio

- `snake.py` — Código fuente del juego con las modificaciones implementadas
- `README.md` — Documentación del proyecto

## 🛠️ Modificaciones realizadas

### Commit 1: Código original
Se incluyó la versión original del juego Snake del paquete freegames como punto de partida del proyecto.

### Commit 2: Movimiento aleatorio de la comida (Rol Owner)
Se agregó la función `move_food()` que permite que la comida se mueva aleatoriamente un paso a la vez en cualquiera de las cuatro direcciones. Se incluyó una validación de límites para evitar que la comida salga de la ventana de juego.

### Commit 3: Colores aleatorios (Rol Fork)
Se cambió el juego para que cada vez que se corra, la serpiente y la comida tengan colores diferentes elegidos aleatoriamente de una lista de 5 colores (azul, verde, naranja, morado, rosa), excluyendo el rojo que se reserva para el indicador de game over.
